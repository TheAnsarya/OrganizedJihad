OrganizedJihad Userscript Setup Guide

Use this page after running the installer. Pick your browser section below and follow the same three-part flow: install Tampermonkey, import organized-jihad.user.js, then verify the script is enabled.

Userscript file location: %LOCALAPPDATA%\OrganizedJihad\userscript\organized-jihad.user.js

Before You Start

  1. Close any old Hero Wars browser tabs before importing the script.
  2. Install Tampermonkey in your browser (links are opened by installer when enabled).
  3. In Tampermonkey, import organized-jihad.user.js from your installed userscript folder.
  4. Confirm the script is enabled, then refresh Hero Wars.

Google Chrome

  1. Install Tampermonkey from the Chrome Web Store.
  2. Open Tampermonkey Dashboard -> Utilities -> Import from file.
  3. Select organized-jihad.user.js and click Install.

Microsoft Edge

  1. Install Tampermonkey from the Edge Add-ons store.
  2. Open the dashboard and import organized-jihad.user.js.
  3. Enable the script and refresh Hero Wars.

Mozilla Firefox

  1. Install Tampermonkey from addons.mozilla.org.
  2. Use Tampermonkey Utilities to import organized-jihad.user.js.
  3. Confirm script status is Enabled.

Opera

  1. Install Tampermonkey from Opera Add-ons.
  2. Open Tampermonkey Dashboard and import the script file.
  3. Enable script, then refresh Hero Wars.

Opera GX

  1. Install Tampermonkey from the Opera Add-ons page (same extension feed as Opera).
  2. Import organized-jihad.user.js from the userscript install folder.
  3. Keep the script enabled for hero-wars.com domains.
